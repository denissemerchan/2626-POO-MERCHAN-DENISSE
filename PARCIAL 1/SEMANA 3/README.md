# Proyecto Gestión de Mascotas

## Estudiante: 
DENISSE MARGARITA MERCHAN JAUREGUI

## Descripción
Proyecto que implementa la gestión de mascotas mediante Programación Tradicional y Orientada a Objetos en Python.

## Reflexión
La programación tradicional es directa y basada en funciones, mientras que la Programacion Orientada a Objetos facilita la reutilización de código y modela mejor objetos del mundo real mediante clases y atributos.
