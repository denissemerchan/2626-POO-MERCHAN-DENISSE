## Descripción del Sistema
Un sistema modular básico desarrollado en Python para gestionar productos y clientes de un restaurante, aplicando principios elementales de la Programación Orientada a Objetos.

## Estructura del Proyecto
- `modelos/`: Contiene las clases de datos (`Producto` y `Cliente`).
- `servicios/`: Contiene la lógica del negocio (`Restaurante`) para gestionar listas.
- `main.py`: Punto de entrada del programa.

## Tipos de Datos Utilizados
- `str`: Nombres de productos, clientes y del restaurante.
- `int`: Asignación de números de mesa.
- `float`: Precios de los platos y bebidas.
- `bool`: Disponibilidad del producto y tipo de cliente.
- `list`: Tipo compuesto para almacenar colecciones de objetos.
